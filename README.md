# CampusThrift
Thrifting menjadi salah satu cara masyarakat memenuhi kebutuhan. Thrifting merupakan kegiatan membeli barang bekas karena keuntungannya dalam menghemat pengeluaran yang tetap mengikuti jaman (Julia, S. R., Zunaedi, R. A., & Putra, P. S., 2024). 

Dalam penerapannya toko thrifting masih banyak yang belum memanfaatkan teknologi dengan baik agar penjualannya maksimal. Proyek kali ini dikembangkan untuk membantu mahasiswa dalam melakukan kegiatan jual-beli dengan sistem wesbite yang interaktif dan mudah digunakan.

Oleh karena itu, pengembangan website dengan framework Laravel sangat membantu. Proyek ini diharapkan dapat menciptakan solusi untuk meningkatkan efektivitas dan memberikan kontribusi yang positif.

## Fitur Utama
### 1. Halaman login/sign up
Sebelum masuk ke halaman home, pengguna diwajibkan login/sign up untuk mendaftarkan diri sebagai pembeli maupun penjual. Sign up mengharuskan pengguna untuk memasukan nama,  membutuhkan validasi email dan password. Log in mengharuskan pengguna memasukan email dan password yang sebelumnya sudah didaftarkan pada saat sign up.
### 2. Halaman Home
Halaman home akan menampilkan daftar produk yang dijual, seperti nama produk, harga, dan ID dari penjual. Pengguna dapat menerapkan filter berdasarkan jenis produk ataupun harga untuk melihat produk yang diinginkan. Pengguna juga dapat memilih apakah ingin menggunakan bahasa Indonesia atau Inggris.
### 3. Halaman Appointment
Apabila pengguna merasa cocok dan ingin membeli produk, pengguna dapat langsung membuat janji dengan penjual dengan memilih tanggal, waktu, dan lokasi bertemu dengan penjual. Pengguna akan memilih tanggal melalui kalender, jam melalui tampilan jam digital, dan lokasi melalui gambaran map.
### 4. Halaman Tambah Produk
Pengguna dapat menambahkan foto produk, nama, deskripsi, harga, jenis, metode bayar, dan lokasi.
### 5. Halaman Jadwal
Pengguna dapat melihat jadwal bertemu dengan penjual/pembeli dalam bentuk kalender dan jadwal dikategorikan sebagai complete (sudah selesai) dan pending (yang akan datang).
### 6. Halaman lokasi
Website akan menampilkan pop up lokasi apa saja yang direkomendasikan untuk melakukan janji temu.
### 7. Halaman detail produk 
Halaman dapat diakses setelah pengguna melakukan janji temu dengan penjual/pembeli dan akan menampilkan gambar, harga, dan id dari transaksi yang akan dilakukan.
### 8. Halaman obrolan
Pengguna dapat mengaksesnya setelah melakukan janji temu dengan penjual/pembeli. Melalui halaman ini, pengguna dapat melakukan obrolan dengan penjual/pembeli demi kenyamanan saat melakukan transaksi jual-beli.
### 9. Halaman profil pengguna
Pengguna dapat mengedit data pribadi, seperti nama depan, nama belakang, nama universitas, dan email. Selain itu pengguna juga dapat melihat kredibilitas dari akunnya, seperti berapa produk yang sudah dibeli, total transaksi, status akun, dan usia dari akun pengguna. Daftar transaksi yang sudah diselesaikan juga dapat dilihat pada halaman ini dan pengguna dapat melaporkan penjual/pembeli yang dianggap mencurigakan, tidak setuju dengan kondisi produk, dan lain sebagainya.

## Tools yang digunakan
Implementasi proyek yang kami kerjakan menggunakan beberapa tools dan teknologi, yaitu:
1. Bahasa pemrograman	: PHP
2. Framework		    : Laravel 11
3. Database		        : mysql
4. Platform		        : Visual Studio Code dan GitHub

## Hasil
Pengembangan website berhasil diimplementasikan dengan hasil sebagai berikut:
1. Website CampusThrift berhasil dibangun dengan tampilan responsif dan mudah digunakan oleh pengguna.
2. Halaman-halaman yang ada dapat menampilkan isi yang sesuai.
3. Fitur localization berjalan dengan baik sehingga pengguna dapat memilih bahasa Indonesia atau Inggris.
4. Database penyimpanan data produk dan pengguna berhasil ditampilkan di website.
5. Sistem berjalan stabil pada seluruh halaman utama, seperti home, login/sign up, detail produk, dan halaman lainnya.

Website dapat digunakan oleh penjual dan pembeli dari mahasiswa dengan baik dan lancar. 

## Kesimpulan
Hasil project website yang telah kami kembangkan sudah mencapai tujuan yang ada, yaitu:
1. Menyediakan website thrift menggunakan framework Laravel sebagai media penjualan produk bekas secara daring.
2. Menyediakan antarmuka website yang mudah digunakan (user-friendly).

Secara keseluruhan website yang ada telah berjalan sesuai kebutuhan pengguna dan siap digunakan sebagai media penjualan online.
